Завдання для самостійної роботи з SQL та PostgreSQL:

---

**Назва завдання:** Робота з базою даних PostgreSQL

**Мета завдання:** Освоєння навичок роботи з базою даних PostgreSQL, написання SQL-запитів та використання інструментів pgAdmin та VScode.

**Інструкції:**

1. **Створення бази даних:**
   - Використовуючи pgAdmin або VScode, створіть нову базу даних PostgreSQL.

2. **Відновлення бази даних:**
   - Відновіть базу даних з файлу `.tar`, який знаходиться в репозиторії GitHub викладача.
   
3. **Написання SQL-запитів:**
   - Напишіть 20 різноманітних SQL-запитів для роботи з цією базою даних. Запити повинні включати:
     - Вибірку даних (SELECT)
     - Фільтрацію даних (WHERE)
     - З'єднання таблиць (JOIN)
     - Групування даних (GROUP BY)
     - Використання агрегатних функцій (наприклад, COUNT, AVG)
     - Сортування даних (ORDER BY)
     
   - Кожен запит має містити коментар, який пояснює його мету та логіку.

4. **Збереження запитів:**
   - Збережіть всі запити у файлі з розширенням `.sql`.

5. **Публікація роботи:**
   - Створіть власний репозиторій на GitHub та викладіть туди файл `.sql` з вашими запитами в окремій паці. 
   Назва  папки - прізвище слухача латинськими літерами.
   - Зробіть `pull request` до репозиторію викладача для перевірки вашої роботи.

**Критерії оцінювання:**
- Правильність створення та відновлення бази даних.
- Коректність та різноманітність SQL-запитів.
- Наявність пояснювальних коментарів у файлі `.sql`.
- Дотримання процедури публікації роботи на GitHub.
